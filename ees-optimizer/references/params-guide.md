# 储能配置优化 - 参数说明

## 功率搜索范围 (MW)

| 参数 | 默认值 | 含义 |
|------|--------|------|
| 功率最小值 | 500 | 网格搜索功率下限 |
| 功率最大值 | 2000 | 网格搜索功率上限 |
| 功率步长 | 100 | 每次递增的功率值 |

功率范围越大、步长越小，搜索组合越多，计算时间越长。

## 容量搜索范围 (MWh)

| 参数 | 默认值 | 含义 |
|------|--------|------|
| 容量最小值 | 2000 | 网格搜索容量下限 |
| 容量最大值 | 10000 | 网格搜索容量上限 |
| 容量步长 | 500 | 每次递增的容量值 |

## 电价参数

| 参数 | 默认值 | 含义 |
|------|--------|------|
| 电价上限 | 2000 | 电价截断上限。超过此值的电价会被替换 |
| 电价下限 | 0 | 电价截断下限。低于此值的电价会被替换 |

## 模型参数（基础）

| 参数 | 默认值 | 范围 | 含义 |
|------|--------|------|------|
| 基准弃电率 (%) | 10.0 | 0~100 | 新能源整体基准弃电率，用于生成逐日弃电率数组，约束储能消纳弃风弃光比例 |
| 充放电效率 η | 0.95 | 0.50~1.00 | 储能系统充放电往返效率（round-trip efficiency） |
| 放电深度 DOD | 0.90 | 0.10~1.00 | 电池放电深度，影响可用容量和循环磨损计算 |
| 容量单位成本 (万元/kWh) | 0.4 | >0 | 每 kWh 储能容量的投资单价。总投资 = 功率×功率单价 + 容量×容量单价 |
| 补全电价数据 | 开启 | 开/关 | 是否对缺失/异常电价做补全处理 |

## 高级参数 — 成本

| 参数 | 默认值 | 含义 |
|------|--------|------|
| 功率单位成本 (万元/kW) | 0.6 | 每 kW 储能功率的投资单价 |

## 高级参数 — 寿命

| 参数 | 默认值 | 含义 |
|------|--------|------|
| 循环寿命 (次) | 2000 | 电池充放电循环次数，用于计算最大允许能量吞吐量（usage_limit） |
| 日历老化率 | 1.0 | 日历老化系数，修正电池容量随时间的自然衰减 |

## 高级参数 — 财务

| 参数 | 默认值 | 含义 |
|------|--------|------|
| 折现率 | 0.07 (7%) | 资金折现率，用于 NPV 和逐年折现现金流计算 |
| 项目年限 (年) | 10 | 项目经济评估周期，影响 NPV 和 IRR 计算 |

## 高级参数 — 求解

| 参数 | 默认值 | 含义 |
|------|--------|------|
| 时间尺度 scale | 0.25 | 时间步长（小时），0.25 = 15分钟一个决策时段 |
| 求解时限 (秒) | 100 | 每个配置组合的 HiGHS 求解器最大运行时间 |
| 磨损单位成本 (元/MWh) | 0.01 | 每 MWh 充放电的磨损惩罚成本 |
| 换电运输成本 (元) | 5.0 | 换电操作的固定运输费用 |
| 计算天数 | 366 | 模拟的天数范围（1~366） |
| 辅助服务类型 | NR | NR=非旋转备用，SR=旋转备用 |

## 数据文件

| 文件 | 格式 | 用途 |
|------|------|------|
| LMP_data.xlsx | .xlsx | 节点边际电价数据（Locational Marginal Price） |
| 风光发电功率.xlsx | .xlsx | 逐时段可再生能源发电量数据 |

项目自带默认数据文件，用户可通过 Web 界面上传覆盖。

## 关键计算公式

- **总投资** = 功率(MW) × 1000 × 功率单价(万元/kW) + 容量(MWh) × 容量单价(万元/kWh) × 1000（万元）
- **最优方案判定** = 单位收益比（年收益 / 总投资）最大
- **usage_limit** = 循环寿命 × 容量 / DOD × 计算天数 / 365
- **简单回收期** = 总投资(元) / 年收益(元)
- **NPV** = -总投资 + Σ(年收益 / (1+折现率)^年)
