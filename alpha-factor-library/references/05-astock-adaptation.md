## 一、A 股市场特殊性

### 1.1 制度差异

| 差异项 | A 股 | 美股(原论文) | 对因子的影响 |
|--------|------|------------|-------------|
| **交易制度** | T+1 | T+0 | Delay-0 因子无法直接使用，必须用 Delay-1 |
| **涨跌停** | ±10%/±20%(创业板/科创板) | 无 | 涨跌停日信号失真，需过滤 |
| **做空限制** | 融券门槛高、券源少 | 融券便利 | 多空组合中空头难以实现 |
| **交易时间** | 9:30-15:00(含午休) | 9:30-16:00(连续) | 日内形态因子含义不同 |
| **停牌** | 频繁停牌(虽已减少) | 罕见 | 停牌日数据需特殊处理 |
| **印花税** | 卖出 0.05% | 无 | 换手率高的因子成本更大 |

### 1.2 市场结构差异

| 差异项 | A 股 | 美股 | 影响 |
|--------|------|------|------|
| **投资者结构** | 散户占比高(~60%交易量) | 机构主导 | 量价行为模式不同，部分因子可能更有效 |
| **小盘股效应** | 显著且持续 | 已大幅衰减 | 需做市值中性化 |
| **行业分类** | 申万行业(31个一级) | GICS/SIC | 替换 IndClass |
| **基准指数** | 中证500/沪深300 | S&P 500 | GTJA191 中基准指数用中证500 |
| **VWAP计算** | = AMOUNT/VOLUME | 交易所直接提供 | 需自行计算 |

## 二、数据处理规范

### 2.1 价格数据

| 处理项 | 方法 | 原因 |
|--------|------|------|
| **复权方式** | **前复权** | 消除除权除息的价格跳变 |
| **VWAP** | AMOUNT / VOLUME | A 股需自行计算 |
| **成交额** | 使用 AMOUNT 字段 | GTJA191 多处使用成交额 |

### 2.2 股票池过滤

| 过滤规则 | 条件 | 原因 |
|----------|------|------|
| **新股** | 上市不足 60 日赋 NaN | 避免新股效应和次新股波动 |
| **ST/\*ST** | 全部剔除 | 制度性风险，交易规则不同(±5%) |
| **停牌** | 停牌日因子值 = NaN，不参与排名 | 数据缺失 |
| **涨跌停** | 涨跌停当日信号不执行交易 | 无法成交 |
| **次日一字板** | 若次日开盘即涨/跌停，跳过 | T+1 无法买入 |

### 2.3 异常值处理

| 步骤 | 方法 | 与你当前管线的兼容性 |
|------|------|---------------------|
| **截断** | Q1%~Q99% 截断 | ✅ 与你现有 Q99% 截断一致 |
| **归一化** | 截面 Z-score 或截面排名百分位 | ✅ 建议用截面排名，更稳健 |
| **缺失值** | NaN 不参与截面排名，不补值 | 避免引入噪声 |

### 2.4 行业中性化

```python

    lambda x: (x - x.mean()) / x.std()
```

### 4.2 预处理 Pipeline

```python
def prepare_alpha_feature(factor_raw, listing_days, is_limit, sw_industry):
    """Alpha因子 → CNNMSV2输入特征的标准管线"""
    
    # 1. 新股过滤
    factor = factor_raw.where(listing_days >= 60)
    
    # 2. 涨跌停过滤
    factor = factor.where(~is_limit)
    
    # 3. Q1%-Q99% 截断
    q01 = factor.quantile(0.01, axis=1)
    q99 = factor.quantile(0.99, axis=1)
    factor = factor.clip(q01, q99, axis=0)
    
    # 4. 行业中性化（申万一级）
    factor = factor.groupby(sw_industry, axis=1).transform(
        lambda x: (x - x.mean()) / (x.std() + 1e-8)
    )
    
    # 5. 截面排名标准化 → (0, 1]
    factor = factor.rank(axis=1, pct=True)
    
    return factor
```

### 5.2 递归因子处理

**Alpha191#143** (SELF递归):
```python
def alpha143(close):
    """递归因子：上涨时乘涨幅"""
    result = pd.Series(1.0, index=close.index)
    for i in range(1, len(close)):
        if close.iloc[i] > close.iloc[i-1]:
            pct = (close.iloc[i] - close.iloc[i-1]) / close.iloc[i-1]
            result.iloc[i] = result.iloc[i-1] * pct
        else:
            result.iloc[i] = result.iloc[i-1]
    return result
```
