# 高级训练技巧

## 一、模型权重平均

| 技术 | 原理 | 使用建议 |
|------|------|----------|
| **EMA（指数移动平均）** | 对训练过程中的权重做指数滑动平均 | 衰减率 0.999~0.9999，训练后期用 EMA 权重评估 |
| **SWA（随机权重平均）** | 对多个 checkpoint 的权重取平均 | 训练后期开启，LR 用恒定值或周期性 |

- EMA 几乎没有额外计算开销，推荐**默认开启**
- SWA 倾向于找到更平坦的 loss 最小值，泛化性更好
- 两者都能对超参数变化提供一定的鲁棒性

## 二、混合精度训练（FP16 / BF16）

| 格式 | 特点 | 适用场景 |
|------|------|----------|
| **FP32** | 标准精度，无精度损失 | 默认/调试 |
| **FP16** | 省显存、加速，但需 loss scaling | NVIDIA GPU（Volta+） |
| **BF16** | 省显存、加速，范围大不易溢出 | A100/H100/TPU |

**实用建议：**
- BF16 优先于 FP16（如果硬件支持），因为不需要 loss scaling
- FP16 训练必须开启 **GradScaler**（自动 loss scaling）
- 保持 **FP32 的 master weight**（混合精度的核心）
- 如果训练出现 NaN，先检查是否是精度问题

## 三、数据增强技巧（用于缓解过拟合）

| 技术 | 原理 | 适用场景 |
|------|------|----------|
| **Mixup** | 两张样本按比例混合，标签也按比例 | 通用，对分类和回归都有效 |
| **CutMix** | 切下一块区域用另一张图填充 | CV 任务 |
| **CutOut** | 随机遮挡一块区域 | CV 任务，简单有效 |
| **Label Smoothing** | 软化 one-hot 标签 | 分类任务，α=0.1 是常见默认值 |
| **特征噪声** | 给输入特征加高斯噪声 | 表格数据 / 时序数据 |

## 四、梯度相关技巧

| 技术 | 用途 | 参数建议 |
|------|------|----------|
| **梯度裁剪（Clip by Norm）** | 防止梯度爆炸 | 阈值 = 梯度 norm 的 90% 分位数 |
| **梯度裁剪（Clip by Value）** | 粗暴但有效 | 通常 [-1, 1] 或 [-5, 5] |
| **梯度累积** | 模拟大 batch | 注意 BN 统计量不变 |
| **梯度监控** | 诊断训练健康度 | 关注梯度 norm 的均值和方差 |
