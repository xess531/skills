## 1. Barra CNE6 多因子风险模型

### 1.1 模型定位

MSCI 开发的业界标准风险模型，用于：**风险归因** → **组合优化** → **Alpha 与 Beta 分离**。

### 1.2 CNE6 因子体系（A 股专用）

| 风格因子 | 英文 | 描述变量 | 经济逻辑 |
|----------|------|----------|----------|
| **Beta** | Beta | 个股对市场的系统风险暴露 | CAPM beta |
| **动量** | Momentum | 相对强弱(RSTR)、历史alpha | 趋势延续 |
| **规模** | Size | 对数流通市值 | 大小盘效应 |
| **盈利** | Earnings Yield | E/P、现金流/价格 | 价值因子变体 |
| **成长** | Growth | 营收增长率、利润增长率 | 成长溢价 |
| **杠杆** | Leverage | 资产负债率、净负债率 | 财务风险 |
| **账面市值比** | Book-to-Price | B/P | 经典价值因子 |
| **波动率** | Residual Volatility | 残差波动率、日内波幅 | 低波异象 |
| **流动性** | Liquidity | 换手率(1M/3M/12M) | 流动性溢价 |
| **非线性规模** | Non-linear Size | log(市值)³ | 中盘股效应 |

### 1.3 Barra 纯因子模型

```
r_i = α_i + Σ(X_ij × f_j) + u_i

其中：
r_i = 资产 i 的收益
X_ij = 资产 i 对因子 j 的暴露度（截面标准化后的因子值）
f_j = 因子 j 的因子收益（截面回归估计）
u_i = 特质收益（个股 alpha 来源）
```

### 1.4 纯因子收益估计

**截面回归法（WLS，加权最小二乘）**：

每个时间截面 t：
```
r_t = X_t × f_t + u_t

权重：√(市值)（大市值股票更稳定）
```

### 1.5 风险归因分解

**组合风险**：
```
σ²_p = w' × (X × F × X' + D) × w
F = Cov(f): 因子协方差矩阵
D = diag(σ²_u): 特质风险对角矩阵
```

**主动风险（跟踪误差）**：
```
TE² = (w_p - w_b)' × V × (w_p - w_b)
    = 因子风险贡献 + 特质风险贡献
```

### 1.6 A 股 Barra 因子特殊现象

| 因子 | A 股表现 |
|------|----------|
| **规模(SMB)** | 长期有效，小盘股溢价显著且持续 |
| **价值(HML)** | 周期性波动，牛市中通常失效 |
| **动量** | 与美股相反——短期反转强于中期动量 |
| **波动率** | 低波异象有效，但受市场环境影响大 |
| **换手率** | A 股特有强因子，低换手率长期跑赢 |

### 1.7 对因子研究的指导意义

- 新因子必须检验与 Barra 风格因子的相关性（独立 alpha vs 已知 beta）
- 行业/市值中性化 = 对 Barra 行业/规模因子做正交化
- 组合优化中用 Barra 协方差矩阵做风险预算

## 2. Fama-French 因子定价框架

### 2.1 模型体系

**FF3**（1993）：r = α + β_MKT × MKT + β_SMB × SMB + β_HML × HML
**FF5**（2015）：增加 RMW（盈利因子）+ CMA（投资因子）

### 2.2 A 股实证结论（2024 年主流研究）

| 因子 | A 股表现 | 说明 |
|------|----------|------|
| MKT | 显著 | 市场因子在任何市场都显著 |
| SMB（规模） | **极显著** | A 股小盘效应持续存在 |
| HML（价值） | **弱于美股** | 定价效率更低 |
| RMW（盈利） | 有效但波动大 | 季度更新频率受限 |
| CMA（投资） | **不显著** | 在 A 股解释力弱 |

### 2.3 与 Barra 的关系

| 维度 | Fama-French | Barra |
|------|-------------|-------|
| 定位 | 学术定价模型 | 风险管理模型 |
| 关注点 | 因子溢价是否显著 | 因子暴露和协方差估计 |
| 实操 | 验证因子是否有 alpha | 做风控和组合优化 |

**两者互补**：FF 验证因子是否有 alpha，Barra 做风控。

## 3. 因子合成方法论

### 3.1 合成方法对比

| 方法 | 公式 | 优点 | 缺点 | 适用 |
|------|------|------|------|------|
| **等权** | f = Σf_i / N | 简单稳健 | 无视因子质量 | <5 个因子 |
| **IC 加权** | w_i = mean(IC_i, N天) | 考虑预测力 | 对 IC 估计敏感 | 5-20 个因子 |
| **ICIR 加权** | w_i = IC_mean_i / IC_std_i | 兼顾预测力+稳定性 | 需足够历史 | 5-20 个因子（推荐） |
| **最大化IC** | max w'μ / √(w'Σw) | 理论最优 | 估计误差放大 | 需正则化 |
| **ML 合成** | XGBoost/LightGBM | 非线性交互 | 过拟合风险 | >10 个因子 |

### 3.2 代码示例

```python
# ICIR 加权
N = 60  # 回看窗口
weights = [mean(IC_i[-N:]) / std(IC_i[-N:]) for i in factors]
weights = weights / sum(abs(weights))
composite = sum(w_i * factor_i for w_i, factor_i in zip(weights, factors))

# 机器学习合成
import lightgbm as lgb
model = lgb.LGBMRegressor(n_estimators=100, max_depth=3, subsample=0.8)
model.fit(X_train, y_train)
composite = model.predict(X_test)
```

### 3.3 因子正交化（去冗余）

| 方法 | 做法 | 效果 |
|------|------|------|
| **施密特正交** | 逐一正交，对前面因子回归取残差 | 简单但顺序敏感 |
| **对称正交** | 联合正交化，保持等距 | **推荐**（无顺序偏差） |
| **回归法** | 因子对其他因子回归取残差 | 同施密特 |
| **PCA** | 提取主成分 | 丢失可解释性 |

**实操建议**：
- <10 个因子：ICIR 加权 + 对称正交
- 10-50 个因子：ML 合成（XGBoost）
- >50 个因子：先 PCA 降维或分组后组内合成

## 4. 因子拥挤度与衰减

### 4.1 因子拥挤度

**定义**：同一因子被过多策略使用 → 溢价被挤压甚至反转。

**MSCI 五维度拥挤度模型**：

| 维度 | 指标 | 含义 |
|------|------|------|
| **估值价差** | Top/Bottom 组估值差 | 极端 → 拥挤 |
| **交易量集中度** | 高暴露组交易量占比 | 异常高 → 拥挤 |
| **因子动量** | 因子近期累积收益 | 短期大涨 → 过度 |
| **成对相关性** | 高暴露股票间收益相关 | 高相关 → 同向交易多 |
| **短期利息** | 高暴露股票融券余额 | 融券量大 → 空头拥挤 |

```python
crowding_score = w1*val_spread + w2*vol_conc + w3*fac_mom + w4*pair_corr + w5*short_int
# 综合分 > 70 → 高拥挤度预警
```

### 4.2 因子衰减 (Alpha Decay)

IC 随持有期增长而下降。衰减速度反映因子信息的时效性：

| 类型 | 半衰期 | 典型因子 | 调仓频率 |
|------|--------|----------|----------|
| **快衰** | < 5 天 | 高频量价、短期反转 | 日频/T+1 |
| **中衰** | 5-20 天 | 换手率、动量 | 周频 |
| **慢衰** | > 20 天 | 基本面、质量 | 月频 |

```python
for horizon in [1, 2, 3, 5, 10, 20, 40, 60]:
    ic = compute_rank_ic(factor, forward_return(horizon))
    print(f"Hold {horizon}d: IC = {ic:.4f}")
```

### 4.3 因子失效的三大原因

1. **拥挤**：信号被充分交易后溢价消失
2. **过度拟合**：样本内显著但样本外无效
3. **体制变化**：市场结构变化导致因子逻辑不成立

**防御策略**：
- 定期监控滚动 IC（60 日均值跌破 0 → 预警）
- 多因子组合分散单一因子失效风险
- 新因子上线前必须做样本外验证

## 5. 交易成本建模 🆕

### 5.1 为什么交易成本对因子研究至关重要

- 高频因子（RankIC 5-8%）的**大部分 alpha 可能被交易成本吞噬**
- 换手率因子效果好但**不等于能实现**：Top 组换手率低=流动性差=冲击成本高
- **不考虑交易成本的回测是虚假的**

### 5.2 交易成本的构成

```
总交易成本 = 显性成本 + 隐性成本

显性成本:
  - 佣金：万 2-3（双边）
  - 印花税：千 1（卖出单边）
  - 过户费：十万分之 2

隐性成本（通常远大于显性成本）:
  - 买卖价差 (Spread Cost)
  - 市场冲击 (Market Impact)：大额交易推动价格
  - 时机成本 (Timing Cost)：执行延迟导致的价格变化
  - 信息泄露 (Information Leakage)：交易意图暴露
```

### 5.3 市场冲击模型

#### 平方根冲击模型（实务中最常用）

```python
# Almgren et al. (2005)
impact = η * σ * (Q / ADV) ** γ

# η: 冲击系数（与市场/标的相关）
# σ: 日波动率（年化波动率 / √252）
# Q: 交易股数
# ADV: 日均成交量
# γ: 幂指数（通常 0.5，即平方根）
```

**A 股参数经验值**：

| 标的池 | η（冲击系数） | γ（幂指数） | 单边冲击（万手） |
|--------|-------------|------------|-----------------|
| **沪深 300** | 0.10 - 0.15 | 0.5 | ~5-10 bps |
| **中证 500** | 0.15 - 0.25 | 0.5 | ~10-20 bps |
| **中证 1000** | 0.25 - 0.40 | 0.5 | ~20-40 bps |
| **微盘股** | 0.50+ | 0.5-0.6 | ~50-100+ bps |

#### Almgren-Chriss 模型（最优执行）

```python
# 最优执行策略：将大单拆分为 N 个子单
# 目标：最小化 E[cost] + λ * Var[cost]

# 临时冲击（即时恢复）
temp_impact = η * σ * (trade_rate / ADV) ** 0.5

# 永久冲击（不可逆）
perm_impact = γ_perm * (total_volume / ADV)

# 总冲击
total_impact = temp_impact + perm_impact
```

**关键洞察**：
- 临时冲击随执行速度↑而↑（急→贵）
- 永久冲击与总交易量成比例（大→贵）
- **最优执行 = 在急切度和冲击成本之间权衡**

### 5.4 交易成本对因子研究的影响

#### 成本感知的因子评估

```python
def net_factor_return(factor, returns, turnover, cost_per_trade=0.003):
    """
    扣除交易成本后的因子收益
    cost_per_trade: 单边交易成本（包含佣金+印花税+冲击）
    """
    gross_return = compute_long_short_return(factor, returns)
    trading_cost = turnover * cost_per_trade * 2  # 双边
    net_return = gross_return - trading_cost
    return net_return
```

#### 因子换手率与净收益的关系

| 因子类型 | 年化换手率 | 估计双边成本 | 扣除成本后 |
|----------|-----------|-------------|-----------|
| 5 日反转 | ~2400% | ~72% 年化 | **可能亏损** |
| 20 日换手率 | ~600% | ~18% 年化 | 仍有正超额 |
| 60 日动量 | ~300% | ~9% 年化 | 大部分保留 |
| ROE 质量 | ~100% | ~3% 年化 | 几乎全部保留 |
| F-Score | ~50% | ~1.5% 年化 | 最佳 cost-效率 |

**结论**：
- **高频因子不一定比低频因子强**——扣除交易成本后，低换手的质量/价值因子可能更优
- 因子评估必须报告**净 Sharpe（扣除交易成本后的 Sharpe）**
- 换手率约束（如限制年化换手率 <600%）应作为因子筛选的硬约束

### 5.5 降低交易成本的实务方法

| 方法 | 说明 |
|------|------|
| **组合换手率约束** | 限制每次调仓的换手上限（如 20%） |
| **信号平滑** | 因子值取 3-5 日均值，降低噪声导致的频繁换仓 |
| **阈值触发调仓** | 排名变化超过阈值才换仓（如只换排名变化 >20% 的股票） |
| **渐进换仓** | 不一次性换到目标持仓，分 2-3 天渐进执行 |
| **VWAP/TWAP 算法** | 大单拆分，跟踪 VWAP 执行 |
| **冲击成本预算** | 为每笔交易设定冲击预算上限 |
